# 🐘 Elephants & Ladders

A whimsical, multiplayer reimagining of the classic Snakes & Ladders game. Built with Next.js 14, Supabase, and deployed on Vercel.

## 🎮 Features

- **Real-time Multiplayer**: Play with friends using Supabase Realtime
- **Animated Game Board**: 10x10 snake-pattern grid with elephant ladders and mud slides
- **Interactive Dice**: Animated dice rolling with Framer Motion
- **Colorful Player Pieces**: Choose your favorite elephant color
- **Game History**: Track recent moves and special events

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| Next.js 14 | React framework (App Router) |
| Supabase | Database, Auth, Realtime |
| Vercel | Hosting & deployment |
| Tailwind CSS | Styling & UI |
| Framer Motion | Animations |
| Zustand | State management |
